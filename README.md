## Running the project locally
1. Clone or download the repository:
```
git clone https://github.com/witalewski/hc-reviews.git
```
2. Install dependencies:
```
cd hc-reviews
yarn
```
3. Start the application:
```
yarn start
```
4. Go to http://localhost:3000/
